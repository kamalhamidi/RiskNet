🏗️ CE QUI A ÉTÉ CONSTRUIT

✅ ARCHITECTURE:
   • Graph Neural Network Hybride (GCN + GAT)
   • 36,929 paramètres
   • Couches: GCN₁(9→64) + GAT(64→64) + GCN₂(64→32) + FC(32→1)
   • Régularisation: Dropout, BatchNorm, Weight Decay

✅ CONSTRUCTION GRAPHE:
   • 395 nœuds (étudiants)
   • 3,950 arêtes pondérées (relations de similarité)
   • KNN adaptatif (K=10) + Pondération Gaussienne
   • Densité optimale: 5.08%

✅ GESTION CLASS IMBALANCE:
   • Weighted BCE Loss (poids par classe)
   • Focal Loss (optionnel)
   • Stratified K-Fold Validation
   • Early Stopping avec patience=20

✅ ÉVALUATION ROBUSTE:
   • Train/Test Split stratifié (80/20)
   • 6 métriques complètes (Accuracy, Precision, Recall, etc.)
   • ROC Curve + Confusion Matrix
   • t-SNE Embeddings Visualization
   • Analysis d'overfitting

❓ FAQ RAPIDES

Q: Mon modèle va-t-il fonctionner sur une autre université?
A: Probablement pas directement (contexte différent).
   Solution: Transfer learning ou simple retraining sur nouveaux data.

Q: Comment prédire pour un nouvel étudiant?
A: Reconstruire le graphe avec historique + l'étudiant nouveau.
   Voir GUIDE_USAGE.txt section 6.2

Q: Pourquoi GAT + GCN et pas juste GAT?
A: GCN = agrégation globale lissée
   GAT = attention sélective locale
   Combinaison = meilleur des deux mondes (validé empiriquement!)

Q: Mes résultats vont-ils se dégrader avec le temps?
A: Possiblement (model drift). Solution: monitor + retrain régulier.
   Voir GUIDE_USAGE.txt section 6.4

Q: Comment gérer un nouveau classe imbalance ratio?
A: Recalculer pos_weight = N_neg / N_pos automatiquement.
   Voir gnn_student_risk.py classe GNNTrainer
